# docker-omekas-web
Example for building Omeka S's Dockfile.

# Build

````
docker-compose build
````

# Run

````
docker-compuse up
````

Open url

````
http://[APACHE-IP]/
````


# Debug

````
docker-compose run app bash
````

# Configuration

- Default database: omekas
- Default database user: omekas
- Default database password: password
